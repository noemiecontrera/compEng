# Engenharia da Computação - 5º Semestre

Repositório destinado ao armazenamento de exercícios, algoritmos e projetos desenvolvidos durante as aulas e estudos autônomos. Aqui você encontrará códigos em C e Portugol (Portugol Webtudio).

### 📂 Estrutura do Repositório ###
Os arquivos estão organizados por tecnologia para facilitar a navegação:

[Lógica de Programação](./Lógica%20de%20Programação): Algoritmos e pseudocódigos desenvolvidos no início do aprendizado.

[Linguagem C](./C): Programas compiláveis, focados em lógica, estruturas de dados e manipulação de memória.

[Linguagem Java](.Java): Projetos e exercícios focados em Programação Orientada a Objetos (POO)

[Linguagem Python](.Python): exercícios iniciais. (Em progresso!)

* 🔍 Aprendizados: Operações matemáticas, variáveis, if/else, for loops, do-while loops, classes, métodos e pilares da POO. 

* 💻 Desafios: Calculadoras, verificadores de números (par/ímpar, primos, maior número), tabuada, contagem progressiva/regressiva.
